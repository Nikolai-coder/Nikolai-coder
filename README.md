## 🎧 Current Project

### DIAVLO WAV

A modern Windows desktop application for downloading, converting and analysing audiovisual content.

Current areas of development:

* Multi-format audio and video downloads
* WAV, FLAC, MP3 and MP4 conversion
* Playlist processing
* BPM and musical key detection
* BPM switch and key switch analysis
* Local processing with FFmpeg
* Native Windows desktop experience
* Dark and minimal audio-focused interface
* Automated Windows release pipeline
